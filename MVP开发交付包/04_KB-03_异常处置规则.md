# KB-03 异常处置规则

## 通用原则

异常处置回答必须采用：

> 先停机安全确认 - 复核数据 - 判断原因 - 调整方案 - 再次检验

智能体只能提供排查顺序，不得虚构设备故障诊断，不得给危险操作建议，不得替代学生最终判断。

## 异常处置表

| 异常情境 | 智能体建议排查顺序 | 学生最终判断依据 | 建议输出物 | 禁止偏离 |
|---|---|---|---|---|
| 孔距偏差接近0.020mm | 暂停进给 -> 复核刀具磨损 -> 复核刀补 -> 检查装夹基准 -> 小量补偿后复测 | 图纸公差、G6实测数据、质检员判定 | 异常处置记录 | 不得直接合格放行；不得跳过复测 |
| 滚压后Ra未达到0.80μm目标 | 检查滚压刀状态 -> 调整进给/压力 -> 复测Ra -> 确认尺寸是否变化 | 粗糙度仪报告、外圆复测数据 | G4滚压效果复核记录 | 不得虚构Ra数据；不得给危险操作建议 |
| 关键轴径向跳动偏大 | 检查双顶尖状态 -> 复核中心孔 -> 重新打表找正 -> 复测 | 百分表读数、G5放行标准 | 跳动复测记录 | 不得替代1号质检员放行 |
| 装配后转动卡滞 | 检查压装深度 -> 齿轮啮合 -> 轴承入位 -> 接触斑/间隙 -> 复测跳动 | G7/G8记录、装配钳工判断 | 装配异常处置记录 | 不得建议强行继续装配 |

## 示例回答结构

用户问：

> 孔距偏差0.018mm，接近0.020mm阈值，请给出安全处置顺序。

建议回答：

| 步骤 | 处置内容 |
|---|---|
| 1. 先停机安全确认 | 暂停进给，避免继续扩大偏差。 |
| 2. 复核数据 | 复核孔距测量方法和G6实测记录。 |
| 3. 判断原因 | 依次复核刀具磨损、刀补和装夹基准。 |
| 4. 调整方案 | 在确认原因后小量补偿，不直接大幅调整。 |
| 5. 再次检验 | 修正后复测并记录异常处置过程。 |

最终判断依据为图纸公差、G6实测数据和质检员判定。
