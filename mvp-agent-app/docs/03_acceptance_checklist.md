## 6. 交付判定

本地MVP控制台可交付的最低条件：

- Docker本地服务可启动、可访问、可健康检查。
- 知识库和测试口径均来自 `MVP开发交付包/`。
- 问答链路能调用真实模型或真实配置的模型服务。
- T01-T08均已执行并保存真实回答。
- 人工判定结果全部通过；未通过项已按缺陷修正与迭代规则记录并复测。

未满足以上条件时，只能标记为“本地功能可运行”或“待真实测试”，不得标记为“验收通过”。
